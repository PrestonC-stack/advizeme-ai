Project Overview

AI-powered estimate-building and DVI analysis system for an automotive repair shop (Callahan Auto & Diesel). The system ingests technician DVIs (Autoflow), repair orders (ROs), and existing estimates, then standardizes, audits, and converts them into clean, complete, sellable estimate structures aligned with TechMetric.

Primary purpose:

Eliminate missed revenue, inconsistencies, and incomplete documentation
Enforce quality control across technicians and advisors
Standardize workflows for scalability
Act as an intelligent bridge between technician findings and advisor sales execution

This system is intended to function as both:

A production tool (building estimates, guiding advisors)
A quality control layer (auditing DVIs, enforcing standards)
Core Objective

Create a system that converts technician findings into structured, complete, audit-verified estimate job blocks while guiding advisors on prioritization, follow-up, and sales actions—ensuring no missed work, no duplication, and consistent high-quality output.

DVI / Inspection Standards

“Good” DVI includes:

Clear, specific findings (not vague)
Supporting photos (before/after, close-up)
Measurable data when possible
Cause + effect explanation
No generic phrases (“needs attention”)

Required:

Description
Evidence (photo or test)
Justification
