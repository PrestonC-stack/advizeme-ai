# AdvizeMe.ai

This folder is the working home for the AdvizeMe.ai project.

Start by placing ChatGPT exports and project notes into the folders below.

- `exports`: dense project handoffs from ChatGPT threads
- `prompts`: master prompts, audit prompts, coaching prompts
- `examples`: good/bad examples, sample DVIs, sample estimates
- `docs`: planning docs, architecture notes, workflow definitions
- `build`: app scaffolding and implementation files
- `assets`: screenshots, diagrams, reference images

- `exports/chatgpt-project-export-01.md`
- `prompts/dvi-audit-prompt.md`
- `prompts/estimate-audit-prompt.md`
- `examples/good-dvi-example.md`
- `examples/bad-dvi-example.md`
- `docs/project-master-context.md`
