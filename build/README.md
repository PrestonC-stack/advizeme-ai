# AdvizeMe.ai Build Workspace

This folder is now reserved for implementation artifacts for the pilot MVP.

## Planned structure

- `desktop/`: lightweight Windows desktop copilot
- `dashboard/`: live workflow dashboard
- `supabase/`: schema, seed data, SQL, and backend notes

## First implementation targets

### `desktop/`

- Electron shell
- React side panel
- Windows notifications
- manual event entry
- targeted capture hooks

### `dashboard/`

- active ticket board
- alert list
- filters for location and advisor
- stale workflow view

### `supabase/`

- initial schema
- seed data for locations and staff
- views for current ticket state and alert state

## Pilot-first rule

If a feature does not help the one-computer pilot on older Windows 10 hardware, it should wait.
