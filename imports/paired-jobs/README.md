# Paired Job Imports

Use this folder when you want one vehicle or one job tied together across systems.

## Suggested structure

Create one file set per vehicle or job:

- `paired-job-01-summary.md`
- `paired-job-01-tekmetric.txt`
- `paired-job-01-autoflow.txt`
- `paired-job-01-trello.txt`

## In the summary file include

- customer or company name
- RO number
- vehicle
- location
- what the issue was
- what got missed, stalled, or needed attention
