## What Still Needs More Samples

### Highest value next uploads

- 1 to 2 AutoFlow DVI screen screenshots, not just PDF exports
- 1 AutoTextMe message-thread screenshot
- 1 to 2 Tekmetric customer-update or communications views
- 2 to 3 real estimate PDFs tied to the same jobs as the DVI PDFs if possible
- 2 to 3 written workflow notes explaining what went wrong or what needed attention

### Most useful pairings

If possible, upload matched sets for the same vehicle:

- DVI PDF
- RO PDF
- Tekmetric screen screenshot
- AutoFlow screen screenshot
- short note about what the advisor had to decide or what got missed

## Classification Notes

- Tekmetric is easy to identify by the dark/orange interface and RO/estimate/job-board views.
- AutoFlow / AutoTextMe is easy to identify by the blue interface and Techflow / calendar / workflow views.
- Trello is clearly separated and useful for Apache-specific workflow tracking.
- The main remaining ambiguity is not system identification anymore. It is matching related files to the same vehicle or job.
